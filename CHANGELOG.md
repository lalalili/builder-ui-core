# Changelog

All notable changes to `@lalalili/builder-ui-core` will be documented in this file.

## v1.0.0 - 2026-07-05

### Added
- `BuilderShell` — Builder 頁面外殼骨架
- `RuleTreeBuilder` 與 `registerRuleTreeBuilder` — 條件規則樹編輯器（Vue 元件與自訂元素註冊）
- 規則樹：AND/OR 巢狀群組、`IN` / `NOT IN` 逗號分隔多值、依 operator 動態 placeholder、
  日期欄位相對日期（`today±N`）
- rule-tree utils 測試（vitest）
